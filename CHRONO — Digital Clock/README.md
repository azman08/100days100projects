# ⏱️ Chrono — Advanced Digital Clock Dashboard

🚀 **Live Demo:**  
(https://100days100projects-hull.vercel.app/)

A **Digital Clock… but built like a real product — not a 30-line demo.**

Most developers build a digital clock while learning JavaScript.  
It usually takes **20–30 lines of code**.

But this project goes much further.

Instead of a basic clock, I built a **fully interactive time dashboard** with animations, utilities, and a modern UI using only:

- HTML
- CSS
- JavaScript

No frameworks.  
No libraries.  
Just **clean frontend engineering.**

---

# 🕐 Core Clock

- Real-time digital clock running at **60fps** using `requestAnimationFrame`
- Smooth **flip-digit animations**
- Digits update **only when values change**
- **12 / 24 hour toggle**
- Optional **seconds display**
- **Live Unix timestamp**
